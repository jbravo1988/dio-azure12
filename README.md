Azure Arc
Ferramenta nos modos de multi-cloud (Azure resource Manager)
O Azure Arc permite o gerenciamento de recursos em ambientes híbridos e multicloud, incluindo servidores e serviços hospedados em AWS (Amazon Web Services), Google Cloud Platform (GCP) e também em data centers locais (on-premises)
Arc não administra as maquinas dentro do Azure e sim as maquinas de fora.
Azure Resource Manager - ARM
Fornece uma camada de gerenciamento que permite criar, atualizar e excluir recursos na assinatura do Azure.
